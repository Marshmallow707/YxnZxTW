## About the Project

This project is a Tailwind CSS version of my original Bootstrap-based website, redesigned with several improvements in layout, styling, and overall user interface. The goal of this project was to explore and apply modern utility-first CSS techniques using Tailwind while recreating and enhancing an existing web design.

The website serves as a personal Otaku-themed blog featuring sections for anime-related content such as news, movies, and series. It was developed as a course project submission for **IT101 - JK01**.
